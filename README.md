# bspwm-dotfiles

### To install setup from scratch 
I would recommend that you go through the script before executing it :)
``` 
wget https://raw.githubusercontent.com/theCode-Breaker/bspwm-dotfiles/main/bspwm.sh 
bash bspwm.sh
```
